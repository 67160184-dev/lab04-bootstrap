# Lab 04: Blog Website with Bootstrap 5

เอกสารนี้จัดทำขึ้นเพื่อแนะนำการใช้งานโปรเจค **Lab 04: Blog Website with Bootstrap 5** ซึ่งเป็นเว็บไซต์ Blog ด้าน IT พัฒนาด้วย Bootstrap 5 และใช้ข้อมูลแบบ Mockup (ไม่เชื่อมต่อฐานข้อมูล)

---

## 1. วัตถุประสงค์

- ฝึกใช้งาน Bootstrap 5 และ Responsive Design
- ฝึกจัดโครงสร้างไฟล์เว็บไซต์
- ฝึกใช้ Git และการ Commit ตามขั้นตอน
- ฝึกใช้ JavaScript สำหรับตรวจสอบฟอร์ม

---

## 2. โครงสร้างไฟล์

```
lab04-bootstrap/
├── index.html        (หน้าแรก)
├── blog-post.html    (หน้าแสดงบทความ)
├── register.html     (หน้าสมัครสมาชิก)
├── login.html        (หน้า Login)
├── create-blog.html  (หน้าสร้างบทความใหม่)
├── css/style.css     (CSS)
├── js/script.js      (JavaScript)
└── images/           (รูปภาพบทความ)
```

---

## 3. วิธีการใช้งานเว็บไซต์

### หน้าแรก (index.html)

- แสดงบทความในรูปแบบ Card
- คลิกปุ่ม **อ่านต่อ** เพื่อดูบทความ
- มีปุ่ม **สร้างบทความใหม่**

### หน้าบทความ (blog-post.html)

- แสดงบทความตาม `id` จาก URL
- ใช้ Mockup Data จาก JavaScript

### หน้าสมัครสมาชิก (register.html)

- ฟอร์มสมัครสมาชิกพร้อม Validation
- สมัครสำเร็จจะไปหน้า Login

### หน้า Login (login.html)

- ฟอร์มเข้าสู่ระบบแบบ Mockup
- Login สำเร็จกลับหน้าแรก

### หน้าสร้างบทความ (create-blog.html)

- ฟอร์มสร้างบทความใหม่
- ตรวจสอบข้อมูลก่อนส่งฟอร์ม

---

## 4. การทดสอบ

1. ตรวจสอบการแสดงบทความหน้าแรก
2. ทดสอบการคลิกอ่านบทความ
3. ทดสอบ Validation ของฟอร์ม
4. ทดสอบการเชื่อมโยงแต่ละหน้า

---

## 5. หมายเหตุ

- โปรเจคนี้ใช้ข้อมูลตัวอย่าง (Mockup)
- ไม่มีการเชื่อมต่อฐานข้อมูลจริง
